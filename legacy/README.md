# Legacy / archived root files

One-off launchers, fix scripts, and old notes that used to sit in the **repository root** are stored here so the top level stays small and obvious.

| Folder | Contents |
|--------|----------|
| `batch/` | Old `.bat` helpers (run/fix/deploy experiments). **Use `RUN_GRADIO_APP.bat` at repo root** or `python quick_start.py`. |
| `python/` | Standalone scratch / reorganizer / fix scripts not wired into the main app. |
| `markdown/` | Old status / guide `.md` files duplicated or superseded by `docs/`. |
| `text/` | Old `.txt` pointers and checklists. |
| `other/` | Odd one-offs (e.g. stray `.ps1` at root, `sample_data.csv` duplicate). |

Nothing here is required to run Meta AI. Canonical paths:

- Run: `quick_start.py` or `RUN_GRADIO_APP.bat`
- Docs: `docs/GETTING_STARTED.md`, `docs/HOW_TO_DEPLOY.md`
